# 车牌识别
## 接口
- 说明：车牌识别
- 路由地址：/lpr
- 请求方式：POST
- Content-Type: application/json
- 参数：

|名称|类型|默认值|描述|
|-|-|-|-|
|image|string|""|base64图片(与file字段互斥)|
|file|string|""|图片文件路径(与image字段互斥)|
